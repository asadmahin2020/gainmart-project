
GainMart - Simple Frontend

This is a basic frontend for an e-commerce website for supplements.
It is made for a university project. It is NOT a real shop.

Tech used:
- React (for pages and components)
- React Router (for page navigation)
- Tailwind CSS (for simple styling)



How to run (for teacher or examiner):

1. Open a terminal in this project folder.
2. Run:  npm install
3. Then run:  npm start
4. Open http://localhost:3000 in the browser.

Pages:
- Home page with hero section and some featured products.
- Products page with product grid and simple filters.
- Product details page with add-to-cart button.
- Cart page with quantity update and remove.
- Checkout page with a simple form and fake order.
- Login and Register pages (only frontend forms).

Note:
- All product data is coming from src/data/products.js.
